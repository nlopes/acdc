# Changelog

## [Unreleased]

### Added

- Experimental PDF converter backend that renders core AsciiDoc documents through Typst,
  including document titles, sections, inline formatting, lists, basic tables,
  admonitions, links, footnotes, table of contents, thematic breaks, and page breaks.
  Media-oriented features such as images, audio, video, and source highlighting currently
  render textual fallbacks with converter warnings.
