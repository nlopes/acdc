FI:: *FI* is the notation for *Future Issue*.
A proposal, decision, or clarification marked with *FI* needs to be addressed.

#<issue-number>:: The number of an active issue that addresses a proposal.
The issue number replaces the *FI* notation on an item.

SDR:: *SDR* is the notation for *Specification Document Requirement*.
Items marked with this notation are terms that we need to define.
Such items may need an issue to resolve their name or definition.
