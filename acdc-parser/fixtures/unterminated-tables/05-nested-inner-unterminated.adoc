= Inner unterminated table inside an `a`-style cell

Outer pipe table terminates normally. The inner `!===` inside the
AsciiDoc-style cell never closes, so the unterminated warning fires for
the **inner** table only:
`unterminated table block (opened by `!===`)`.

The outer table renders normally and the inner is a nested table whose
boundary is the end of the cell's content slice (not end of document).

The warning resolves to the line of the inner `!===`, not the cell's
`a|` style prefix line.

[cols="1a"]
|===
a|
!===
! Inner A ! Inner B
|===

Paragraph safely outside the outer table.
