# `acdc-term`

A simple terminal parser for `AsciiDoc` documents.

## Usage

```bash
acdc-term samples/simple.adoc
```

![Simple Document](images/simple.adoc.png)

You can also pass multiple files and it will parse and print them all.

```bash
acdc-term samples/*
```
