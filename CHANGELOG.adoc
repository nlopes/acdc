= Changelog

This project uses per-crate changelogs. See the relevant crate for its changelog:

== Parser

- link:acdc-parser/CHANGELOG.md[acdc-parser]: An AsciiDoc parser.

== Tools

- link:acdc-cli/CHANGELOG.md[acdc-cli]: This is the main binary for the command line processor. Think of it as the equivalent to the `asciidoctor` tool.
- link:acdc-lsp/CHANGELOG.md[acdc-lsp]: A Language Server Protocol (LSP) implementation for AsciiDoc documents, powered by link:acdc-parser[].
- link:acdc-editor-wasm/CHANGELOG.md[acdc-editor-wasm]: A WebAssembly-based AsciiDoc live editor with syntax highlighting and HTML preview.
